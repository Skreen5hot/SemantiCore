Edge-canonical semantic enrichment

SemantiCore

Spec v1.0

Structured data in. Deterministic JSON-LD graphs out.

SemantiCore will enrich CSV, JSON, and JSON-LD records with local TagTeam.js semantic graphs while preserving the edge-canonical rule: browser or Node, no required infrastructure.

Phase 0

Repository activation and app runway are in progress.

Workbench Shell

This static page is the GitHub Pages publication target. The functional local-first workbench will be built in later phases around the pure SemantiCore kernel.

Input

CSV, JSON, or JSON-LD adapters normalize records into canonical JSON-LD.

Pathing

Semantic property paths select string literals independent of JSON-LD compaction.

TagTeam

A pinned local TagTeam runtime produces named graph JSON-LD for each selected value.

State

IndexedDB and filesystem persistence remain adapters, never kernel requirements.
